# Chain Drop

A small mobile-friendly chain puzzle game built with plain HTML, CSS, and JavaScript.

## Play locally

Open `index.html` in a browser, or serve this folder with any static web server.

## Customize blocks

Edit `block-assets.js` and place block images in `assets/blocks/`.

## Customize sidekick

Edit `character-assets.js` and place expression images in `assets/characters/`.
The config maps expression names, such as `idle` and `cheer`, to image files,
then maps game states to those expressions.

## Customize sound effects

Edit `audio-assets.js` and place sound files in `assets/sounds/`. Any event
with a blank `src` keeps using the built-in synthesized sound.

## GitHub Pages

Publish the repository root from the `main` branch.
