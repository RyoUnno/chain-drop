# Chain Drop

A small mobile-friendly chain puzzle game built with plain HTML, CSS, and JavaScript.

## Play locally

Open `index.html` in a browser, or serve this folder with any static web server.

## Customize blocks

Edit `block-assets.js` and place block images in `assets/blocks/`.

## Customize sidekick

Edit `character-assets.js` and place character images in `assets/characters/`.

## GitHub Pages

Publish the repository root from the `main` branch.
